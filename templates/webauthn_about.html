Use Passkeys / Security Keys with VeriQuantum

  1. Create an account (Register) and log in once with password.
  2. Open Settings → Security and choose Add Security Key (WebAuthn).
  3. Follow the browser prompt to register a device (Face/Touch ID, YubiKey, etc.).
  4. Next time, choose Login with Passkey to authenticate passwordless.

Passkeys are phishing-resistant and protected by your device’s secure hardware.
